SQUASH

A SQuaRE service for the verification datasets group.

Purpose

SQUASH provides a minimum infrastructure for the verification datasets group to carry out QA analysis on precursor datasets with the LSST stack. As a prototype SQUASH can be used and extended by the verification scientists, while preserving the code and knowledge developed by this group in a centralized place.

How to Use

Start by selecting a dataset registered in SQUASH and check the existing features, then learn how to extend the system.
