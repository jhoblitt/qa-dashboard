SQUASH

A QA service developed by SQuaRE.

Purpose

SQUASH provides a minimum infrastructure for testing the LSST stack using precursor datasets. As a prototype SQUASH can be used and extended by the LSST scientists preserving the code and knowledge developed during construction.

How to Use

Start by looking at the metrics reported by KPM CI and check the existing features, and help SQuaRE to extend this system.
